Avalon

A system to manage parts inventory for aircraft based in New Zealand

About avalon

Avalon is a Windows program that manages parts for aircraft maintenance.  It was specifically designed and developed in conjunction with Heli Maintenance Ltd., Christchurch New Zealand.

The Avalon database can be based on MS ACCESS or MS SQL Server.  

features

  • Issuing of parts to a job number
  • Manage part numbers with release note ID
  • Alternative Part Numbering
  • Receipt of parts into stock
  • Stock level reporting
  • Shelf Life reporting
  • Inventory valuation
  • Labelling
  • Job cost reporting
